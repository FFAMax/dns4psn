# DNS4ALL
## Requirements
Python3.x, pip, twisted

## Quick start
> Example for Debian 9.6

```bash
apt-get update
apt-get install curl python3-twisted dnsutils git
git clone https://github.com/FFAMax/dns4psn.git
cd dns4psn/
python3 dns.py
```
